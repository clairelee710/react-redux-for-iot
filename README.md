## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/clairelee710/react-redux-for-iot.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
### Key of this project   

React, Redux, ES5, JSX, Front-End, IoT
